# 🚀 TempoSync: Automated B2B Invoice Reconciliation (Serverless)
**TempoSync** is a serverless, automated B2B invoice reconciliation platform built natively on the **Tempo Testnet**. It solves the biggest accounting hurdle for Web3 businesses: matching incoming crypto payments to specific customers and invoices.

## ✨ Why Tempo? 
* **Native Transfer Metadata:** We embed invoice IDs directly into stablecoin transactions.
* **Stablecoin-Native Gas:** Merchants and customers deal purely in USD denominated stablecoins.
* **Deterministic Finality (0.5s):** Payments are confirmed instantly.
